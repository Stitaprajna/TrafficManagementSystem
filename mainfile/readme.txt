
########################### Manual Instructions to run the code ##########################

1. Download the zip-file and extract it 
2. Open the whole folder in any code editor
3. Open the terminal in the code editor and run:
    pip install -r requirements.txt 
    cd Main/
4. Put your video inside this folder and change the input path in main.py
5. Run main.py
6. If you are not seeing the fonts properly, then change the font-size, color, location. 
The default pixel size choosen is 1080px720p. 
7. Detail images of speed-tracking regions is also given in this file. 
Go through the main.py file understand the project in a better way.
8. Output will be saved in out.mp4, you can change the format also.
